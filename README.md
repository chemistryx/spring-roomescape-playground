# 🔐 방탈출 예약 관리 어플리케이션

## 예약 API 사용법

### 예약 추가 (POST /reservations)

#### 요청 예시

```json
{
  "name": "브라운",
  "date": "2025-06-30",
  "time": "15:40"
}
```

#### 응답 예시

```json
{
  "id": 1,
  "name": "브라운",
  "date": "2025-06-30",
  "time": "15:40"
}
```

#### 예외 상황

| 상황                           | 상태              | 예외 메세지                                     |
|------------------------------|-----------------|--------------------------------------------|
| 예약에 필요한 데이터가 빈 값이거나 null일 경우 | 400 Bad Request | `예약하기 위한 데이터(이름, 날짜, 시간)를 모두 입력해 주세요.`     |
| 주어진 날짜 형식에 맞지 않을 경우          | 400 Bad Request | `날짜(년도-월-일)형식에 맞게 입력해 주세요. ex) 2020-12-31` |
| 주어진 시간 형식에 맞지 않을 경우          | 400 Bad Request | `시간(시:분)형식에 맞게 입력해 주세요. ex) 15:30`         |
| 과거의 날짜와 시간에 예약 할 경우          | 400 Bad Request | `이미 지난 날짜 및 시간은 예약할 수 없어요.`                |
| 삭제하려는 예약이 존재하지 않을 경우         | 400 Bad Request | `예약 ID가 존재하지 않아요.`                         |

### 예약 조회 (GET /reservations)

모든 예약 목록 반환

### 예약 삭제 (DELETE /reservations/{id})

예약 ID로 삭제
